## Hascal Features:
1. Cross Platform (Linux, Windows,MacOS)
2. Fast & Powerful
3. Simple syntax (combination of Pascal , JavaScript and Golang)
4. Can be use in web development(can compiles to javascript)
5. DMD back-end
